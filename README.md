# Формування заявок на роботи у електроустановках
## Запуск

Запуск застосунку

  $ npm start

Заповнити базу даними для демонстрації

  $ npm run demo-init

* Типи електрообладнання
* Категорії електробезпеки
* Механізми
* Посада
* Розміщення
* Працівники
